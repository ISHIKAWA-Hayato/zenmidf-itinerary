# zenmidf-itinerary

行程表アプリのプロトタイプ（MVP）です。  
スプレッドシートより使いやすい「1日1ページ（1d1p）」の行程表UIを目指しています。

## ドキュメント
詳細な仕様・開発メモは docs ブランチ側にあります。  
- 詳細 README: docs/README.md（docs ブランチ）
- JSON スキーマ: docs/schema_v0.json（docs ブランチ）

## このリポジトリでやること（概要）
- 1d1p タブ切替の表UI
- JSON 入出力
- PDF 出力（表＋自動タイムライン）
